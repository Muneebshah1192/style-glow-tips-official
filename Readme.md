# Style & Glow Affiliate Store

A modern, responsive, and beautiful affiliate marketing website created for women’s beauty, fashion, accessories, and lifestyle products.  
This website is designed to send Pinterest traffic to product pages and then redirect visitors to Amazon affiliate product links.

## Website Purpose

Style & Glow Affiliate Store helps promote selected beauty and fashion products in a clean, attractive, and user-friendly way.  
The site allows visitors to explore products, view images, read descriptions, and click affiliate links to visit Amazon.

## Main Features

- Beautiful women-focused beauty and fashion design
- Fully responsive layout for mobile, tablet, and desktop
- Smooth animations and interactive product cards
- Product categories for beauty, fashion, accessories, skincare, watches, and more
- Admin panel for managing products
- Add products with image URL or uploaded image
- Edit and delete existing products
- Product search inside admin panel
- Ad Manager for website ads
- Supports custom banner ads
- Supports Google AdSense code placement
- Floating side ad option
- Featured homepage ad option
- Local browser storage for products and ads
- Simple admin login with password and CAPTCHA
- Netlify-ready static website

## Admin Panel

The website includes an admin panel where products and ads can be managed.

### Admin Login

Use the login details that are already configured inside the website files.

For privacy and safety, do not publicly write your admin email or password in this README file.

## Important Security Note

This is a static website made for GitHub and Netlify hosting.  
The admin panel uses front-end protection and browser localStorage.

This is good for testing, demo, and small personal use, but it is not the same as a real server-side secure admin system.

For professional business use, connect the website with one of these:

- Firebase
- Supabase
- Netlify Identity
- MongoDB with backend
- Any secure custom backend
