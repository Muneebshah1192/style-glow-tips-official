Style & Glow Affiliate Website - Pro Admin + Upload + Ad Manager Version

Developed by Muneeb Haider
Pinterest: Style & Glow - Style & Glow Tips Official

How to use:
1. Upload this whole folder to GitHub.
2. Connect the GitHub repo with Netlify.
3. Open /admin/index.html to access the admin panel.
4. Login with your private admin credentials and solve the CAPTCHA.
5. Add products using either image URLs or uploaded images from your device.
6. Edit/delete products from the Running Products section.
7. Manage homepage/side ads from the Ad Manager section. You can use custom banner ads now and paste Google AdSense code after Google approves your site.

Important:
- This is a static Netlify-friendly website.
- Products and ad manager data are saved in browser localStorage.
- Uploaded images are compressed before saving to reduce lag.
- For real multi-device database and true server-side security, connect Firebase, Supabase, or Netlify Identity later.
